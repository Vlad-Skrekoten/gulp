//How start using gulp?

1.  min version of npm and node 
    npm -v && node -v
    8.11.0
    v16.16.0

2.  git clone https://github.com/your-username/your-project-name.git

3.  cd your-project-name 
    Ex:(cd gulp)

4.  npm i

5. Building a distribution version
    gulp docs

    Running project in the development mode
    gulp dev
